# feat: 实现基础日历结构与页面布局

---

## 功能描述

本次 PR 实现了项目的基础页面结构与月历视图组件。

功能作用：
- 构建整体页面布局框架，包含顶部标题区、日历区、日程列表区、语音输入区
- 实现月历视图渲染，支持月份切换
- 支持日期选择与高亮显示

使用方式：
1. 打开页面即可看到当前月份日历
2. 点击左右箭头切换上/下月
3. 点击日期可选中，选中日期显示浅蓝色背景
4. 今天的日期显示蓝色渐变背景

## 测试方式

### 测试步骤

1. 打开页面，验证日历正确显示当前月份
2. 点击左侧箭头，验证月份切换到上一月
3. 点击右侧箭头，验证月份切换到下一月
4. 点击任意日期，验证选中状态变化
5. 验证今天的日期有特殊高亮样式

### 验收标准

- 日历正确显示当月所有日期
- 月份切换功能正常
- 日期选择功能正常
- 页面布局响应式适配移动端
