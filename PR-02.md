# feat: 实现自然语言解析模块

---

## 功能描述

本次 PR 实现了自然语言解析核心模块，支持从用户输入中提取日期、时间与事件信息。

功能作用：
- 识别中文日期表达（今天/明天/后天/下周X/具体日期）
- 识别时段表达（上午/下午/晚上/X点X分）
- 自动提取事件内容

使用方式：
1. 在文本输入框输入自然语言，如"明天下午三点开会"
2. 系统自动解析为结构化日程数据
3. 解析结果显示在识别结果区域

## 实现思路

### 技术选型

| 技术 | 用途 |
|------|------|
| JavaScript | 核心解析逻辑 |
| 正则表达式 | 关键词匹配与提取 |

### 核心实现逻辑

1. 意图识别
```javascript
// 删除意图
if (/删除|取消|移除/.test(text)) → action: 'delete'

// 查询意图  
if (/查看|有什么|查询/.test(text)) → action: 'query'

// 默认添加意图
→ action: 'add'
```

2. 日期提取
```javascript
function extractDate(text) {
    if (/今天/.test(text)) return formatDate(today)
    if (/明天/.test(text)) return formatDate(today + 1)
    if (/后天/.test(text)) return formatDate(today + 2)
    if (/下周([一二三四五六日])/.test(text)) → 计算下周X日期
    if (/(\d+)月(\d+)日/.test(text)) → 解析具体日期
}
```

3. 时间提取
```javascript
function extractTime(text) {
    // 匹配 "下午3点"、"上午10点"、"晚上8点"
    // 下午/晚上 + N点 → 转换为24小时制
    // "下午3点" → 15:00
    // "晚上8点" → 20:00
}
```

4. 事件提取
```javascript
// 移除时间词、日期词后的剩余文本即为事件名
cleanText = text
    .replace(/今天|明天|后天|下周./g, '')
    .replace(/上午|下午|晚上|\d+点/g, '')
    .trim()
```
